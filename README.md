# GigaGrok Web

> Claude.ai-like interface powered by Grok 4.20 beta + memory search

![Next.js](https://img.shields.io/badge/Next.js-14-black?style=flat&logo=next.js)
![Grok](https://img.shields.io/badge/Grok-4.20-purple?style=flat)

## Deploy to Vercel (1 click)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/wojciechkowalczyk11to-tech/gigagrok-web&env=XAI_API_KEY&envDescription=Your%20xAI%20API%20key%20from%20console.x.ai)

Set env var: `XAI_API_KEY=xai-your-key`

## Features
- Grok 4.20 reasoning / fast / code models
- Web search + X search
- Memory search (ULTIMATE_MEGA_MEMORY collection)
- Dark cyberpunk UI
- Streaming responses

## Local dev
```bash
npm install
echo "XAI_API_KEY=xai-..." > .env.local
npm run dev
```
